# Website Update Requests

This file is your running list of things you want changed, fixed, or added on the
website. Keep it open and add to it whenever you think of something — then hand
the whole list to Claude when you're ready.

## How to use this file

1. Open this file any time: `G:\church\UPDATES.md` (double-click it, opens in Notepad).
2. Under **Pending Requests** below, add a new line for each thing you want changed.
   Keep it short — a date and a one-sentence description is enough, for example:
   - `2026-07-12 — Change Sunday service time to 11am`
   - `2026-07-12 — Add a new event for the church anniversary on Aug 15`
   - `2026-07-12 — Upload the 3 photos I put in the new-photos folder`
3. Save the file (Ctrl+S).
4. Open this Claude Code chat and just type something like: "check UPDATES.md and
   do the pending items." I'll read the list, do each one, and tell you when done.
5. Once something is finished, I'll move it down into **Completed**, with the date,
   so you have a history of everything that's been done.

## Adding new photos

If you want new photos added to the website gallery, drop the image files into
the `new-photos` folder (`G:\church\new-photos\`) and add a line under Pending
Requests like `Upload photos from new-photos folder`. I'll upload them and then
clear that folder out.

(Alternatively, you can always upload photos yourself anytime via the admin
panel: riversoflivingwaterchurch.org/admin → Gallery tab → Upload New Image.)
